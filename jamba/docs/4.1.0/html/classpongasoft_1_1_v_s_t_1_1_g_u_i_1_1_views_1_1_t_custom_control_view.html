Detailed Description

template<typename T>
class pongasoft::VST::GUI::Views::TCustomControlView< T >

Base class which extends CustomControlView to provide the type (T) of the underlying parameter this view is managing.

Because the type is known, this class takes care of registering the parameter (using its id given by getControlTag()) and automatically handles vst, jmb or no parameter at all (case when the id is undefined).

See also
Params::GUIOptionalParam for more details on optional parameters
