Detailed Description

template<typename TView, typename TGUIPluginState>
class pongasoft::VST::GUI::Views::PluginView< TView, TGUIPluginState >

When implementing a View specific to a given plugin, you can use this class instead to get direct access to the state and parameters registered with the plugin via the fState/fParams member.
